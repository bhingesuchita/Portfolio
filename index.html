Projects

Spatio-temporal dynamic functional connectivity analysis of fMRI data

Independent vector analysis |Graph theoretical analysis | Markov analysis| MATLAB | GroupICA for fMRI Toolbox|SPM

  • Dynamic functional connectivity analysis studies the networks that are functionally associated and continuously changing over the scanning period.
  • Most techniques examine dynamic functional connectivity of spatial networks while assuming that the spatial networks are stationary.
  • Spatio-temporal dynamic functional connectivity analysis assumes variability in both spatial and temporal domain.
References:
  • S. Bhinge, Q. Long, V. D. Calhoun, and T. Adali, "Spatial Dynamic Functional Connectivity Analysis Identifies Distinctive Biomarkers in Schizophrenia," Frontiers in Neuroscience, vol. 13, 2019.
  • S. Bhinge, R. Mowakeaa, V. D. Calhoun, and T. Adali, "Extraction of time-varying spatio-temporal networks using parameter-tuned constrained IVA," IEEE Transaction on Medical Imaging, vol. 38, issue 7, pp. 1715-1725, 2019.

Abandoned object detection from multi-view video sequences

Independent component analysis | Independent Vector Analysis|Principal Component Analysis | MATLAB

  • The use of multiple cameras for video surveillance has prevailed over the use of a single camera due to the potential for object occlusion.
  • Transposed independence vector analysis jointly analyzes multiple cameras for detection of abandoned objects.
References:
  • S. Bhinge, Y. Levin-Schwartz and T. Adali, "Data-driven fusion of multi-camera video sequences: Application to abandoned object detection," in International Conference on Acoustics, Speech and Signal Processing (ICASSP), New Orleans, LA, 2017, pp. 1697-1701.

Non-orthogonal constrained independent vector analysis

Data fusion|Constrained Optimization | MATLAB

  • In many practical applications important prior information about the data exists and incorporating this information into the model is expected to yield improved separation performance.
  • Independent vector analysis (IVA) provides a flexible and effective approach for the fusion of multiple sensor data.
  • Non-orthogonal constrained IVA incorporates prior information for efficient interaction of data from multiple sensors.
Reference:
  • S. Bhinge, Q. Long, Y. Levin-Schwartz, Z. Boukouvalas, V. D. Calhoun and T. Adali, "Non-orthogonal constrained independent vector analysis: Application to data fusion," in International Conference on Acoustics, Speech and Signal Processing (ICASSP), New Orleans, LA, 2017, pp. 2666-2670.
